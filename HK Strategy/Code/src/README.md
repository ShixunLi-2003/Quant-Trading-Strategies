# Source Tree

This directory contains the runnable source tree used by the command-line scripts.

- Core modules live directly under `analysis/`, `backtests/`, `data/`, `factors/`, `signals/`, and `visualization/`.
- `hk_quant/` is a lightweight compatibility namespace that maps the original project imports to the package-local modules in this folder.
- The annotated review-oriented copies remain in `../framework/`.
