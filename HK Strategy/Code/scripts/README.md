# Scripts

Reproducible command-line scripts used to run backtests and generate diagnostics.

- `run_vectorbt.py`: runs the active cross-sectional Hong Kong strategy.
- `analyze_strategy_stages.py`: exports representative regime windows and stage diagnostics.
- `run_factor_analysis.py`: runs the factor-analysis workflow used in the package.
- `optimize_bear_defense_narrow.py`: parameter sweep utility for the bear-overlay configuration.
- `run_backtrader.py`: event-driven reference backtest for the Backtrader example configuration.
