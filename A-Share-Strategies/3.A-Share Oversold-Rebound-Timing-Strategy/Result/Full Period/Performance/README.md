# Performance

This folder contains the original full-period equity curve, the retimed equity curve, their CSV counterparts, and summary tables used in the package overview documents.

- `equity_curve.csv` / `equity_curve.html`: original portfolio NAV path and interactive chart.
- `retimed_equity_curve.csv` / `retimed_equity_curve.html`: equity-timing overlay output from the original framework.
- `performance_summary.csv` / `retimed_performance_summary.csv`: source metrics reused throughout the package.
- `nav_comparison.png` / `drawdown_comparison.png`: static presentation charts generated directly from the source CSV outputs.
